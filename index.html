Kernel-Level Systems Programming — xv6

Systems • OS

Extended the xv6 teaching OS with production-like shell, scheduling, and memory management features.

  • Added login authentication, process history, syscall block/unblock, and file permission support.
  • Implemented signal handling (terminate, suspend, resume, custom handlers).
  • Designed a priority-boosting scheduler with profiling of turnaround, waiting, and response times.
  • Built memory extensions: per-process memory usage, swapping with adaptive replacement, page faults.
C xv6 OS Internals Scheduling Virtual Memory

Advanced Networking Projects

Networks

A set of projects combining transport protocols, SDN, and server design with real-world inspired behavior.

  • Implemented reliable UDP with TCP-like ACKs, retransmissions, and Reno & CUBIC congestion control.
  • Built SDN controllers (Learning Switch, Spanning Tree, Shortest Path, Congestion-Aware Routing) in Ryu.
  • Designed multithreaded TCP client–server with FIFO/RR scheduling and collision protocols (Slotted Aloha, BEB).
Python Ryu Mininet UDP/TCP SDN

Mini Web Browser

Systems • UI

A lightweight multi-tab browser built from scratch in C++ with a custom stack for fetching, parsing, and rendering.

  • Multi-process/threaded architecture for concurrent fetch, parse, and render of HTML pages.
  • Custom HTML parser that builds a DOM tree for a restricted tag set.
  • Rendering using low-level Qt widgets and QPainter (no built-in web engines).
  • HTTP fetching via libcurl with local caching for repeated loads.
C++ Qt libcurl DOM Rendering

Networks & System Security Lab

Security

End-to-end security exercises covering exploits, network traffic analysis, and TLS/website security.

  • Built buffer overflow, code injection, and return-to-libc attacks on a C web server and implemented fixes.
  • Used sublinear-space algorithms (e.g., Bloom filters) to detect port scans, DDoS, and IP hopping in logs.
  • Developed a Python TLS client for handshake and certificate validation.
  • Performed vulnerability assessment with Nmap, Burp Suite, and Metasploit.
C Python Exploitation Bloom Filters TLS
